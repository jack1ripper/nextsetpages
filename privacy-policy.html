Next Set / 下一组

隐私政策

最近更新：2026 年 5 月 2 日

下一组（Next Set）是一款围绕训练过程管理设计的 iPhone 健身训练应用。当前版本不要求注册账号，也不会将你的训练记录上传到开发者服务器。

1. 我们会处理哪些数据

你在应用内创建或产生的数据可能包括训练动作与计划、训练历史记录，以及应用内偏好设置。这些数据默认仅保存在你的设备本地，用于让应用正常运行。

2. 这些数据如何被使用

  • 展示你的今日训练计划
  • 记录训练完成情况
  • 保存历史记录
  • 保留你的个性化设置

3. 通知与提醒

如果你授权通知权限，下一组（Next Set）可能会在训练过程中向你发送本地通知，用于提醒组间休息结束。这些通知仅在你的设备本地触发，不会用于广告、营销或跨应用跟踪。

4. 第三方共享

当前版本的下一组（Next Set）不会将你的训练数据出售、出租或共享给第三方。

5. 数据存储

你的训练计划、历史记录与偏好设置默认保存在设备本地。如果你删除应用，本地数据可能会一并被删除。

6. 儿童隐私

下一组（Next Set）不面向儿童定向提供服务，也不会主动收集儿童个人信息。

7. 政策更新

如果未来产品能力发生变化，例如增加云同步、账号系统、健康数据联动或第三方服务接入，我们会同步更新本隐私政策。

8. 联系我们

如果你对本隐私政策有任何问题，可以通过支持页面或 GitHub 仓库 Issue 联系开发者：github.com/jack1ripper/fitness/issues

本页面可直接部署到 GitHub Pages，并将公开链接填写到 App Store Connect。
